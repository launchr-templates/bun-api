# bun-api

A lightweight Bun API server.

## Features

- TypeScript support
- Clean, minimalist API design
- Automatic port discovery

## Installation

To install dependencies:

```bash
bun install
```

## Usage

To run in development mode with hot reloading:

```bash
bun dev
```

To run in production mode:

```bash
bun start
```

Or run directly:

```bash
bun run index.ts
```
